# Task Manager: Gestor de Tareas Interactivo

Aplicación web para la organización de tareas personales, enfocada en la productividad y la experiencia de usuario.

## 🚀 Overview
Un gestor de tareas moderno que permite a los usuarios organizar su día a día de forma visual y eficiente, integrando estados dinámicos y persistencia local.

## ✨ Key Achievements
- **Interfaz Reactiva**: Desarrollo de componentes altamente interactivos utilizando React y estados globales fluidos.
- **Persistencia de Datos**: Implementación de almacenamiento local para garantizar que la información esté disponible entre sesiones.
- **User Engagement**: Uso de micro-animaciones y feedback visual inmediato para mejorar la experiencia de productividad.

## 🛠️ Technologies
- TypeScript / React
- CSS Modules / Tailwind
- State Management (Context/Hooks)

## 📦 Features
- Creación y categorización de tareas.
- Seguimiento de estados (Pendiente/Completado).
- Diseño limpio y libre de distracciones.
